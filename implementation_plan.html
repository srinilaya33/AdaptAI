üß© Four-Module Architecture

1

Core Tutor

Goal: Diagnose thinking & answer at student's level

  • Diagnose student thinking - Understand their current level
  • Answer at understanding level - Adaptive complexity
  • OCR for handwritten doubts - Upload images, get explanations
  • Proficiency model - Updates level, picks next question difficulty
  • Personalized XAI - Explains WHY certain hints are given (builds trust)
Smart Adaptation: Language + complexity automatically adapted to student's proficiency level
2

Depth Tools

Goal: Deep research & knowledge exploration

  • Citation Quick Window - Card with title, authors, year, citations
  • Abstract preview - Quick actions: PDF, snapshot, DOI, bookmark
  • Mindmap This - Build concept graph from conversation
  • Attach links to nodes - Wikipedia, citations, educational resources
  • Coding Agent - Run tests, show failures, suggest fixes
  • Web Agent - Search docs/StackOverflow, grounded in mindmap context
Context-Aware: Web searches are filtered by your current mindmap, keeping research focused
3

Visual & Challenge Layer

Goal: Dynamic visualizations & daily practice

  • User Choice - "See visualization" vs "Read explanation"
  • 3Blue1Brown-style animations - For math/algorithms concepts
  • Clear learning goals - Each animation has specific takeaway
  • Daily Challenge Engine - User-specified problems per day
  • Escalating difficulty - Based on current proficiency
  • Spaced repetition - Review weak topics at optimal intervals
Personalized: Tracks preference to avoid overwhelming students with too many visuals
4

Planning & Self-Regulation

Goal: Exam-focused AI study planner

  • Inputs - Exam date, available time, target (pass vs rank)
  • Syllabus analysis - Extract topics from PDF using Textract
  • Past paper weighting - Identify frequently asked topics
  • Auto-create schedule - Daily plan from now to exam
  • Smart slots - Challenges, review weak topics, rest days
  • Visual learning path - Shows progress & what's next
Data-Driven: AI planners outperform manual planning by 30%+ in time management

  • Bedrock Batch API - 50% savings vs on-demand for MVP/Scaled tiers
  • Removed AWS Kendra - Replaced with DynamoDB semantic search (saves ~$800/month)
  • EC2 Spot Instances - 70% cheaper than on-demand for video rendering
  • Maximize Free Tier - Lambda (1M free), DynamoDB (25GB free), API Gateway (1M free)
